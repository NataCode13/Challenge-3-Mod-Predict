# Challenge-3-Mod-Predict

## Modelado predictivo de cancelaciones

## Objetivo
Construir un modelo predictivo para identificar los factores que explican la evasión de clientes y proponer acciones de retención basadas en datos.

## Principales hallazgos esperados
Los factores más asociados al churn tienden a ser:
- contratos mes a mes,
- menor antigüedad,
- internet por fibra óptica,
- pago por electronic check,
- ausencia de soporte técnico y seguridad online.

## Resultado esperado
Un pipeline reproducible que permite anticipar clientes con mayor riesgo de cancelación y apoyar decisiones de retención comercial.
